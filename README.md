# PriCoSha_Project

PriCoSha a system for privately sharing content items
among groups of people that i made for the Databases Project. For this project,
I used
* Python Flask
* MySQL

Team Member
------------------------------------------------

Feature
------------------------------------------------
* View Public Content
* Login
* View shared content items and info about them
* Manage tags
* Post a content
* Tag a content item
* Add friend
* Register
